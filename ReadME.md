# MyDiary
This is an online application that helps users to record their memories.

## Prerequisites
* Text editor
* Browser 

## Features(UI)
* signup page
* signin page
* Diary entries
* Diary entry
* Add and modify entry
* Profile page
* Dashboard page

## BUilt with
* HTML
* CSS
* javascript
